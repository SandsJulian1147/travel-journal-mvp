# 旅行复盘游记工作台离线版 MVP

这个版本把项目重心从“照片地图相册”调整为“旅行复盘和游记生成”。

这是一个本地优先版本：直接打开 `index.html` 即可使用。部署到 Vercel 后，也可以用家庭密钥进入共享旅行空间，开启云端保存和自动同步。

## 新定位

照片不再是主角，而是素材补充。系统的核心作用是引导用户回顾一次过往旅行，把零散记忆整理成：

- 出行动机
- 关键轨迹节点
- 重要事项
- 高光回忆
- 感官线索
- 旅行后的感悟
- 一篇可以继续修改的游记草稿

## 当前 MVP 能力

- 新建和管理过往旅行档案。
- 用“复盘会议”议程引导填写内容。
- 记录路线节点，而不是记录每张照片。
- 新增轨迹节点会插入到最上方，减少长列表滚动。
- 新增轨迹节点会自动沿用已有节点的日期，方便补录多年以前的旅行。
- 在轨迹节点里记录从 A 点到 B 点的交通方式。
- 在轨迹节点里记录地图坐标，用于生成真实轨迹串联。
- 在轨迹节点里上传照片，优先读取 JPEG 照片 EXIF 中的 GPS 定位；没有定位时保留手动坐标兜底。
- 用一个精简字段记录衣食住行、花费、天气穿着、体力和路线提醒。
- 根据坐标生成离线轨迹图、地点/照片节点、动态连线、坐标点数量和大致直线距离；点击地图节点可回到对应记录。
- 记录跨节点的重要事项，如意外、踩坑、经验提醒。
- 记录高光片段、意外转折、关键人物、感官记忆和照片素材备注。
- 自动生成一篇连续的旅行日记草稿。
- 本机自动保存。
- 可选 Vercel Blob 云端保存和自动同步。
- 导出完整记录 JSON。
- 导入 JSON 备份恢复记录。
- 导出游记 Markdown。
- 打印游记，或通过系统打印功能另存为 PDF。

## 本地使用方式

1. 打开 `index.html`。
2. 点击“新建过往旅行”。
3. 按复盘会议议程填写内容。
4. 点击“生成草稿”。
5. 用“导出游记”保存 Markdown。
6. 用“导出记录”保存完整 JSON 备份。

## 数据保存方式

默认使用浏览器本机存储，内容不会上传。配置 Vercel Blob 后，会在本机保存的同时自动同步到云端。

需要注意：

- 同一个浏览器再次打开会自动恢复。
- 如果清理浏览器缓存，记录可能丢失。
- 如果换电脑或换浏览器，先用“导出记录”生成 JSON，再在新环境里用“导入备份”恢复。
- Markdown 是单篇游记文本，JSON 是完整可恢复备份。
- 照片只在本机保存压缩缩略图；自动定位目前优先支持带 EXIF GPS 的 JPEG，截图、微信转存图和多数 PNG 通常没有原始定位信息。

## Vercel 云端同步

1. 在 Vercel 项目里进入 Storage。
2. 创建或连接一个 Vercel Blob store。
3. 确认项目环境变量里有 `BLOB_READ_WRITE_TOKEN`。
4. 重新部署项目。
5. 打开工具首页，在“进入我们的旅行空间”里输入家庭密钥。
6. 点击“进入家庭空间”。

说明：

- 家庭密钥不会明文上传，会先在浏览器里转成 SHA-256 哈希。
- 同一个家庭密钥会进入同一个云端记录空间。
- 家庭密钥要自己保存好，建议 12 位以上。
- 这个版本会把一份完整 JSON 记录保存到 Vercel Blob。
- 这个版本适合个人 MVP 和小范围试用，不是严格账号体系。
- 公开产品阶段建议升级为正式登录、用户表和更细粒度的数据权限。

## 为什么这样改

之前的版本更像“照片地图系统”，适合展示去过哪里和照片在哪里。但用户真正需要的是一套帮助自己理清经历、找回记忆、形成表达的工具。

新的产品主线应该是：

过往旅行经历 -> 复盘会议 -> 结构化记录 -> 游记草稿 -> 人工修改 -> 个人旅行档案。

照片、地图和路线后续作为辅助证据和素材库接入。

## 后续迭代

### 0.2：过往游记增强

- 增加章节重排。
- 支持多种游记风格。
- 支持把照片作为素材拖入某个轨迹节点。
- 支持按交通、餐饮、住宿、总花费自动生成旅行总结。
- 接入地图服务，把地点名称自动转换成坐标。
- 支持从旧地图相册 MVP 导入地点和照片信息。

### 0.3：AI 辅助

- 根据用户填充内容生成更自然的长文。
- 自动提问缺失信息。
- 把流水账改写成叙事游记。
- 生成“给未来自己的总结”。

### 0.4：旅行中记录

- 每日快速记录。
- 语音转文字。
- 当天高光/花费/提醒。
- 旅行结束后一键进入复盘会议。

### 0.5：出发前计划

- 出行目标。
- 行程草案。
- 重要准备事项。
- 期待清单。

### 1.0：完整旅行档案系统

- 计划、途中记录、旅行后复盘串联。
- 地图和照片作为素材层。
- 每次旅行生成完整个人游记。
- 支持按年份、城市、主题回看自己的旅行变化。
- 做线上版本
